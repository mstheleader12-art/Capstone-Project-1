# My Resume Website

## Overview
This is a personal resume website developed using **HTML5** only. The website presents my personal and academic information in a simple, clean, and well-organized format. This project was created to strengthen my understanding of HTML and its fundamental elements.

## Features
- Profile photo
- Objective statement
- Education details
- Work experience
- Skills
- Awards and Certificates
- Hobbies
- Contact information
- Footer section

## Technologies Used
- HTML5

## Purpose
The main purpose of this project is to practice HTML by creating a complete resume website without using CSS or JavaScript. It demonstrates the use of headings, paragraphs, images, tables, lists, hyperlinks, horizontal rules, and footer elements.

## Future Enhancements
- Add CSS for an attractive and responsive design.
- Use Flexbox and Grid for better page layout.
- Add JavaScript to make the website interactive.
- Improve accessibility and user experience.
